# Optimasi K-Means Menggunakan Firefly Algorithm

Repositori ini merupakan implementasi algoritma K-Means yang dioptimasi menggunakan Firefly Algorithm
untuk clustering data multivariat.

## Cara Menjalankan
1. Install dependency:
   pip install -r requirements.txt
2. Masuk ke folder src
3. Jalankan:
   python main.py
